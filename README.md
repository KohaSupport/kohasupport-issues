# KohaSupport Issues

This repository is dedicated to issue tracking for KohaSupport resources and tools.

## About

This is a public repository for tracking bugs, feature requests, and updates related to:
- [KohaSupport Resources Website](https://kohasupport.github.io/kohasupport-resources/)
- Z39.50 Server Directory
- Community tools and utilities

**Note:** This repository contains only issue tracking. The actual code and content are maintained in private repositories for security reasons.

## Issue Types

- **Bug Reports**: Problems with existing functionality
- **Feature Requests**: Suggestions for new features or improvements
- **Server Updates**: Changes to Z39.50 server information
- **Documentation**: Improvements to help or documentation

## Community Guidelines

- Be respectful and constructive
- Provide clear, detailed descriptions
- Search existing issues before creating new ones
- Use appropriate labels and templates
